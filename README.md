# 💳 Wallet Sandbox

Aplicación de billetera virtual full stack desarrollada con React, Node.js, Express y MongoDB.

Permite a los usuarios:
- Registrarse e iniciar sesión
- Consultar su saldo
- Enviar dinero a otros usuarios
- Ver historial de transacciones (enviadas y recibidas)

---

## 🚀 Tecnologías utilizadas

### Frontend
- React (Vite)
- Axios
- CSS

### Backend
- Node.js
- Express
- MongoDB (Atlas)
- JWT (autenticación)
- bcrypt

---

## 🔐 Funcionalidades principales

- Autenticación de usuarios (login/register)
- Transferencias entre usuarios
- Validaciones de seguridad
- Historial de movimientos
- Diferenciación entre envíos y recepciones
- Manejo de estados y UX
